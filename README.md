# Media Save Bot (Telethon, для Termux)

Скачивание видео с YouTube/TikTok/Instagram/X/VK (файлы до 2 ГБ, без лимита
обычного Bot API), поиск музыки по команде **найти <название>**, распознавание
голосовых через Groq Whisper, определение трека в скачанном видео через Shazam.

## Установка в Termux

```
pkg update && pkg upgrade -y
pkg install python ffmpeg git -y
pip install -r requirements.txt
```

## Настройка

1. Скопируй `.env.example` → `.env`
2. Впиши `API_ID` и `API_HASH` с **my.telegram.org** (раздел "API development tools")
3. Впиши `BOT_TOKEN` от **@BotFather**
4. Впиши `GROQ_API_KEY` с **console.groq.com/keys** (бесплатно)
5. (Необязательно) положи рядом файл `cookies.txt`, экспортированный из
   реального YouTube-аккаунта — поможет с блокировками YouTube

## Запуск

```
python bot.py
```

При первом запуске попросит войти — это нормально (Telethon авторизует сессию
бота через MTProto). Дальше просто оставь команду работать в фоне (например,
через `tmux` в Termux, чтобы не завершалось при закрытии приложения).

## Использование

- Пришли ссылку на видео — бот покажет качества на выбор
- Напиши `найти <название трека>` или `трек <название>` — бот найдёт и пришлёт песню
- Пришли голосовое или кружок — бот расшифрует речь в текст
- После скачивания видео бот попробует определить, какая песня в нём звучит,
  и предложит скачать её отдельно кнопкой
